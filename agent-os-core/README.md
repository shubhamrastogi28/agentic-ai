# Agent OS Core

A production-inspired platform for building, orchestrating, and safely executing AI agents.

## Why this project exists
Most AI agent demos focus on prompts.
This project focuses on **agent runtimes**, **orchestration**, **memory**, and **secure tool execution** - the hard platform problems.

## Core Principles
- Agents are stateless
- Memory is external
- Execution is orchestrated, not loop-based
- Horizontal scaling is first-class

## Core Concepts
- Deterministic agent execution using graphs
- Event-driven orchestration
- Pluggable memory backends
- Secure, sandboxed tool execution
- Observability-first design
